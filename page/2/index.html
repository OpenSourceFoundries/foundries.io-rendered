Continuously updated

When you build your products using our microPlatforms you have access to continuously updated software allowing you to rapidly fix security vulnerabilities, deliver bug fixes and add new features

Upstream

The further you get away from the upstream code, the harder it is to keep up. We provide tools and processes to make it easy to develop embedded and connected products while staying close to the upstream projects

Secure

Secure boot, storage and updates provided with fully encrypted communications. OTA updates provided as needed to address CVEs and bugs, and to offer new features

Continuous Integration

Source code is continually rebased and retested against upstream projects to make sure any regressions are quickly found and fixed.

Maintain

Open Source releases and low cost lifetime secure update subscriptions with no lock-in offer lifetime product maintenance without risk

Easy to use

Our microPlatforms deliver key open source technologies as an end-to-end integrated and tested foundations on which to deliver your applications. Simple tools and integration accelerate your product development
